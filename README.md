# pubg-clean-countries-observer
Clean squared countries flag for PUBG teams.

## What is it
PUBG allows to custom teams name and logo, impacting the killfeed and spectating mode. I made my own custom teams with squared and clean countries.

## How to install (Windows)
1. Download and unzip, then copy with `Ctrl + V` (copy) or `Ctrl + X` (move) the `Observer` folder
2. Go to `C:\Users\[USER]\AppData\Local\TslGame\Saved` where `[USER]` is your windows user name
4. Paste and restart the game, this should look like `C:\Users\[USER]\AppData\Local\TslGame\Saved\Observer`
5. You can find online tutorials on how to do it
> You can access `C:\Users\[USER]\AppData\Local\TslGame\Saved` by pressing `Window + R`, typing `%localappdata%` then navigating to `Saved`
